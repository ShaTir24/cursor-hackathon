# EduReels Design System (Master)

## Brand
EduReels — personalized educational video for students and teachers.

## Color
| Token | Hex |
|-------|-----|
| primary | #0B6E4F |

| danger | #B91C1C |
| accent | #E8F5EF |

## Typography
- Display: Fraunces (headings)
- Body: Source Sans 3
- Icons: Lucide

## Dual persona (same palette, different chrome)
| Role | Header eyebrow | Primary CTA | Stage emphasis |
|------|----------------|-------------|----------------|
| Student | Learn | Generate learning reel | Topics + interests → watch |
| Teacher | Teach | Generate class demo | Subjects + grades → watch + lesson pack Sheet |

Do **not** invent a second color system. Differentiate with copy, nav, and actions.

## Layout signature
Video-first workspace: left sidebar navigation, main stage for HLS player and generation actions. Cards for role/onboarding. Teacher lesson pack in a Sheet.

## Motion
150–200ms color/opacity/transform transitions. Respect prefers-reduced-motion.

## Anti-patterns
No purple gradients, no emoji icons, no oversized glassmorphism, no Comic/Baloo playful defaults.
