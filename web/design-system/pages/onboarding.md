# Onboarding (`/onboarding`)

**Job:** Capture display name + catalogue prefs in a clear sequence.

**Layout:** ReUI Stepper — (1) Name + age/grades, (2) Topics/subjects + interests chips, (3) Review + submit. Back between steps; submit only on last (`onboarding-submit`).

**API:** Same `fetchCatalogue` / `submitOnboarding` payloads. Chip multi-select, not dropdowns.
